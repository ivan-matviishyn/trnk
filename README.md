<h1 align="center">Caps config</h1>

## Installation

Clone this repository and install modules:

```bash
git clone https://github.com/ElementalsWeb/caps-config.git
cd caps-config
npm install
```

## Commands

Run development mode

```bash
npm run start
```

Run build mode

```bash
npm run build
```
